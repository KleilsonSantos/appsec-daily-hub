## 📝 Descrição

**Hub Centralizado de Recursos de Segurança de Aplicações** - Uma interface moderna e responsiva que centraliza as principais fontes de informação sobre cibersegurança, vulnerabilidades, ferramentas OWASP e conteúdo educacional para profissionais de AppSec.

## 👀 Visão Geral do Projeto

AppSec Daily Hub é uma aplicação web moderna desenvolvida para proporcionar acesso rápido e organizado aos principais recursos de segurança de aplicações. O projeto adota um design dark theme com foco em usabilidade e experiência do usuário.

### 🏗️ Arquitetura e Organização

O projeto está estruturado de forma modular e responsiva:

• **Interface Moderna**: Design dark theme com glassmorphism e animações interativas
• **Categorização Inteligente**: 6 seções organizadas por tipo de recurso (Notícias, CVEs, OWASP, Podcasts, Favoritos, Social)
• **Responsividade Total**: Mobile-first design com breakpoints otimizados
• **Performance Otimizada**: CSS inline, favicon SVG e carregamento otimizado
• **Acessibilidade**: Navegação keyboard-friendly e contraste adequado

### 🔗 Integração Funcional

- **Curadoria de Conteúdo**: Links verificados e atualizados para as principais fontes de segurança
- **Badges Informativos**: Identificação visual de categorias e tipos de recursos
- **Hover Effects**: Animações suaves e feedback visual em tempo real
- **Status Indicators**: Indicadores visuais de disponibilidade e monitoramento
- **Cross-Platform**: Compatibilidade total com todos os dispositivos e navegadores

### 🚀 Diferenciais da Aplicação

- **Foco em Segurança**: Especializado em recursos de AppSec e cibersegurança
- **Design Profissional**: Interface moderna que reflete o ambiente de segurança
- **Facilidade de Uso**: Navegação intuitiva e acesso rápido aos recursos
- **Atualização Constante**: Conteúdo mantido sempre atual e relevante

> 💡 **Resumo**: O AppSec Daily Hub oferece uma experiência centralizada e profissional para acessar os principais recursos de segurança de aplicações, com design moderno e funcionalidades otimizadas.
